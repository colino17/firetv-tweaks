# Fire TV Tweaks

Various tweaks, scripts, and tricks for Amazon FireTV sticks (tested with Fire OS 7.2.2.3).

## Developer Options

- Turn on ADB debugging (this activates network debugging)
- Allow installing from unknown sources

## Apps to Install (Store)

- Disney Plus
- Youtube
- Netflix
- WWE
- Downloader
